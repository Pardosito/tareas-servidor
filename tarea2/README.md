# Chatsitos Ian

Este proyecto es una pequeña aplicación de chat en tiempo real pensada para practicar conceptos básicos de aplicaciones web y comunicación por sockets. Además, se hizo uso de IA para generar y aplicar estilos para mejorar el aspecto visual de la pequeña aplicación.

## Descripción
- Permite a varios usuarios entrar en diferentes salas de chat.
- Cada usuario elige un nombre, selecciona una sala y puede enviar y recibir mensajes en tiempo real.
- Las conversaciones se actualizan al instante para todos los participantes de la misma sala.

## Características
- Lista de salas disponibles.
- Chats en tiempo real entre los participantes de una sala.
- Avisos de cuando alguien entra o sale de una sala.
- Mensajes muestran la hora y se distinguen mensajes propios del resto.

## Cómo ejecutar
1. Instala dependencias:

   npm install

2. Ejecuta la aplicación en modo desarrollo:

   npm run dev

3. Abre el navegador en:

   http://localhost:3000

4. Ingresa un nombre, selecciona una sala y comienza a usar chatsitos Ian.
